Introduction

This site is a demo of a motel system with customer and admin experiences. It is built with Angular, ASP .NET and PostgreSQL, deployed to Azure AKS using Kubernetes and Helm, with CI/CD via GitHub Actions. For the frontend, React demo is also available. Source code can be seem on GitHub .
